# WebRTC Demo

[English Version](./README_EN.md)

## 项目简介

这是一个基于 Vue 3 和 WebRTC 技术构建的实时视频通话演示项目。本项目展示了如何使用现代 Web 技术实现点对点的视频通讯功能。

## 特性

- 基于 Vue 3 的现代化前端框架
- 使用 Vite 构建工具，提供极速的开发体验
- 集成 WebRTC 技术，实现实时视频通话
- 使用 PeerJS 简化 WebRTC 的使用
- 支持点对点视频通话

## 技术栈

- Vue 3 - 渐进式 JavaScript 框架
- Vite - 下一代前端构建工具
- PeerJS - WebRTC 封装库
- WebRTC - 实时通信技术

## 快速开始

### 安装依赖

```bash
npm install
```

### 开发环境运行

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

### 预览生产构建

```bash
npm run preview
```
